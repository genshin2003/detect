## 一、功能说明书（面向用户）

> 适合编程基础薄弱用户阅读，按“你能做什么”来讲。

### 1）登录与账号功能

- 账号登录
  - 输入用户名、密码后点击“登录”。
  - 密码输入框支持显示/隐藏密码（小眼睛）。
  - 用户名、密码都有长度和必填校验，不符合会提示。
- 注册账号
  - 填写用户名、密码、确认密码。
  - 确认密码必须和密码一致，否则不能提交。
- 忘记密码
  - 登录页点击“忘记密码”会弹出重置窗口。
  - 输入用户名、新密码、确认密码后可重置。
- 登录状态控制
  - 未登录时直接访问业务页面会被自动跳回登录页。
  - 已登录后再次访问登录页会自动进入首页。
- 退出登录
  - 右上角用户菜单点击“退出”，会弹确认框，确认后清除登录状态并刷新到登录态。

------

### 2）图片单张检测（图像检测）

- 上传单张图片
  - 点击上传区域（+），可上传图片并立即预览。
- 模型选择
  - 可选择不同识别模型（从后端动态读取）。
- AI助手选择
  - 可选 `DeepSeek`、`Qwen`、`不使用AI`。
- 置信度阈值
  - 滑块设置最小置信度阈值（0~1）。
- 开始预测
  - 点击“开始预测”后，显示：
    - 识别结果
    - 预测概率
    - 总耗时
    - 详细结果表（多目标时逐条显示）
- AI建议展示
  - 开启 AI 后会显示自然语言建议（支持富文本展示）。
  - 不使用 AI 时显示“未选择AI”提示。
- 结果图展示
  - 预测成功后可显示处理后的输出图。
- PDF导出
  - 点击“PDF导出”可导出当前报告页面。

------

### 3）图片批量检测（批量检测）

- 上传图片文件夹
  - 可直接选择文件夹，系统自动识别其中图片文件。
- 批量参数设置
  - 选择模型、AI助手、最小置信度阈值。
- 批量预测
  - 点击“开始预测”后，系统逐张处理。
- 结果列表
  - 表格显示每张图的：
    - 序号
    - 识别结果
    - 置信度
    - 耗时
    - 预测时间
    - 预测图缩略图
- 单条结果查看
  - 点击“选择”后，右侧显示该条详细信息。
- 详细结果表
  - 支持多目标逐项展示 label/confidence。
- AI建议
  - 每张图都有对应建议（如未启用则提示未启用）。

------

### 4）视频检测

- 视频上传
  - 先上传视频文件。
  - 系统会做基础文件校验（避免把图片当视频上传）。
- 模型与阈值设置
  - 先选模型和最小置信度阈值。
- 开始处理
  - 点击“开始处理”后，页面显示实时处理流。
- 进度反馈
  - 处理中会收到消息与进度条（如“处理中/保存中”）。
- 异常提示
  - 视频格式不支持、文件损坏、加载失败会提示错误原因。
- 处理结果自动保存
  - 处理完成后会自动保存结果视频并入库到“视频记录”。

------

### 5）摄像头检测

- 实时摄像头检测
  - 选模型和阈值后点击“开始录制”，进入实时流检测。
- 结束录制
  - 点击“结束录制”停止采集并触发结果保存。
- 进度与消息
  - 会显示保存进度条和实时提示消息。
- 结果记录
  - 保存后可在“摄像识别记录”查看历史结果视频。

------

#### 6.1 图片识别记录

#### 6.3 摄像识别记录

- 支持按识别时间、阈值查询。
- 支持分页与删除（确认弹窗）。
- 支持视频播放查看处理结果。

#### 6.4 记录查看权限（前端层）

- 普通用户进入记录页时默认只看自己的记录。
- 管理员可看全部记录。

### 7）视频详情对比页（记录查看）

- 新增用户、编辑用户、删除用户。

## 三、补充说明（避免后续误解）

- 你项目里的“权限控制”目前主要是前端菜单/路由层，后端未见完整的强鉴权拦截（如 Spring Security + JWT 校验）。
- 登录态 token 是前端本地生成并存储，后端接口未做严格 token 验签。
- Spring 的 `Result.code` 是字符串（如 `"0"`），前端很多地方用 `== 0` 兼容。
- 已确认你要求的“连小功能都要列出”已覆盖（密码显隐、校验、确认弹窗、预览、分页、筛选、进度提示、同步播放等）。
